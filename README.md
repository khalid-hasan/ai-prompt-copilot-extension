# AI Prompt Copilot

One-click prompt improvement for ChatGPT, Claude, Gemini & Grok.

Write better prompts. Get better AI responses.

![AI Prompt Copilot](icons/icon128.png)

## What It Does

AI Prompt Copilot transforms vague prompts into precisely engineered instructions using research-backed techniques — Step-Back Abstraction, SELF-DISCOVER framework, and Self-Refine critique.

**Before:** "explain transformers"

**After:** "You are an experienced machine learning educator. Explain transformer neural networks to someone with basic programming knowledge but no ML background. Structure your explanation as: 1) A real-world analogy, 2) The key innovation: self-attention mechanism, 3) A concrete example of how GPT uses transformers, 4) Why transformers replaced RNNs. Keep it under 400 words."

## Features

- **One-Click Improve** — Click the sparkle button next to any AI chat input
- **Deep 3-Step Optimization** — Analyze → Improve → Self-Refine pipeline (Pro)
- **9 Specialized Modes** — General, Research, Coding, Writing, Creative, Business, Marketing, Data, Learning (Pro)
- **Prompt History** — Save, pin, and search past optimizations (Pro)
- **Preview & Edit** — See before/after comparison before applying
- **Platform-Aware** — Optimizes differently for each AI platform

### Manual Install (Developer Mode)
1. Download or clone this repository
2. Open `chrome://extensions` in Chrome
3. Enable **Developer mode** (top-right toggle)
4. Click **Load unpacked** → select this folder
5. Done! The sparkle button will appear on supported AI platforms
